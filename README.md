# online_food_delivery_app
🍔 QuickBite Food Delivery Website
A simple and responsive food delivery website built using HTML, CSS, and JavaScript. Users can browse menu items, add them to the cart, and place orders with a dynamic total calculation.

🚀 Features
Responsive design

Interactive food menu

Add to cart functionality

Real-time total price update

Order confirmation message

🛠️ Technologies Used
HTML5

CSS3

JavaScript

📌 How to Use
Open the project folder

Run index.html in your browser

Add items to cart and place your order
